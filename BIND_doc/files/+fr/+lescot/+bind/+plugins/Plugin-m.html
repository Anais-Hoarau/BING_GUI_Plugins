public static isInstanciable

function out = isInstanciable()

This method returns a boolean value that indicates if the plugin is instanciable or not.  This methods purpose is to facilitate research of plugins within the path.  The default value is false, and have to be overwritten in “real plugins”, i.e. plugins that can be used directly, and not only via an heritage.

thisThe object on which the function is called, optionnal.
