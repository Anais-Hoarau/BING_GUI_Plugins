handle
fr.lescot.bind.configurators.Argument

This class represents an argument of a Configuration.  This argument can be an optionnal argument or not.  If it is an optionnal argument, it name will be used in the invocation of the plugin, in order to respect the key / value syntax used by Matlab for such args.  The value can be a single string (under the form of a 1x1 cell array) or a cell array of strings.
