fr.lescot.bind.processing.signalProcessors.LinearInterpoler

fr.lescot.bind.processing.SignalProcessor
fr.lescot.bind.processing.signalProcessors.LinearInterpoler

This processor allows to calculate the value of points at a time not recorded in the signal, using a linear interpolation.  So, using an input signal shaped like this

timecodes [ 1 | 2 | 3 | 4 | 5]
values    [ 0 | 6 | 4 | 5 | 3]

asking for the values at times 1.5, 2 and 2.5 would return :

timecodes [ 1.5 | 2 | 2.5 ]
values    [ 3   | 6 | 5   ]
