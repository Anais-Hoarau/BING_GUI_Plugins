This program will load the data from the CACC experiment and
convert them into the SQLiteTrip format used by BIND software.
The video files will be converted as well.

==============
 Installation
==============
Install the Matlab Runner programm provided in the .zip file.

========================
 How to run the script?
========================
Just launch the CaccImportScripts.exe file.

Warning: Before running this program, make sure that you have
installed:
+ all the necessary codecs to read the videos
+ virtual dub in c:\vdub\ directory (the script will look for c:\vdub\vdub.exe)
+ the divx compression codec
